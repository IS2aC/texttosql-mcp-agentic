DATABASE CONTEXT
---------------------------------------
This database tracks sales of various products in different countries, linking each sale to a specific product and country through foreign keys, enabling analysis of sales data by product, country, and date.

---------------------------------------
DATABASE SCHEMA
---------------------------------------
## Database Schema

**public.countries**
- id integer NOT NULL
- name varchar(100) NOT NULL

**public.products**
- id integer NOT NULL
- name varchar(150) NOT NULL
- category varchar(100)
- unit_price numeric(10,2) NOT NULL
- created_at timestamp without time zone

**public.sales**
- id integer NOT NULL
- product_id integer NOT NULL (FK → public.products.id)
- country_id integer NOT NULL (FK → public.countries.id)
- quantity integer NOT NULL
- sale_date date NOT NULL
- total_amount numeric(12,2) NOT NULL
